Pagina web solo con BOOSTRAP:
La presente pagina web esta diseñanada unicamente con componentes y estilos de BOOSTRAP, utilizando:
 "scrollspy"  --> para permitir que el usuario sepa en que parte se encuentra de la pagina
componentes poropios de navbar y footer de boostrap
"componente modal" --> para la compra de tickets
"carousel de imagenes" --> para la secuencia de imagenes
"bagdes" --> para resaltar cierta pequeña informacion en cada imagen de los oradores
"tooltip" --> para definir y/o redirigirnos a otro vicnculo
"form-row" --> para ingresar un pequeño formulario con 3 inputs
La pagina web es responsive
